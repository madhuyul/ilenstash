# Ilenstash

## Setup Instructions for Windows

1. **Prerequisites**
   - Ensure you have [Git](https://git-scm.com/) installed on your machine.
   - Install [Node.js](https://nodejs.org/) (which includes npm).

2. **Clone the Repository**
   ```bash
   git clone https://github.com/madhuyul/ilenstash.git
   cd ilenstash
   ```

3. **Install Dependencies**
   ```bash
   npm install
   ```

4. **Run the Application**
   ```bash
   npm start
   ```

5. **Open your browser**
   - Navigate to `http://localhost:3000` (or the specified port in the configuration).

Enjoy using Ilenstash!
